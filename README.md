SWT-based browser
=================

Instruction
-----------
+ download the latest stable version of [SWT](http://www.eclipse.org/swt/)
+ copy `swt.jar` into `lib` directory
+ sbt compile
+ sbt console
+ scala> SWTBrowser.main(null)
+ enjoy!
